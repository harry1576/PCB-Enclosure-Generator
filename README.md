# PCB-Enclosure-Generator
A OpenSCAD file that generates 3D printable boxes and lids for PCBs.

### Features:
- Fully configurable.
- Easy to use - just input PCB dimensions.
- Generates stand offs for underside components and solder joints.
- Minimal sharp edges.
- Uses OpenSCAD (Free)
